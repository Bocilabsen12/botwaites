AI LUKY Auto Generate Image

Fitur baru:
- Kalau user menyuruh AI membuat gambar lewat chat biasa / !ai, bot langsung generate gambar dan mengirim file gambarnya ke WhatsApp.
- Tetap kompatibel dengan command !img <prompt> lama.

Contoh:
!ai buatkan gambar kucing putih lucu memakai jaket biru
!ai generate gambar cyberpunk city malam hari
buatkan gambar poster game horror
bikinkan ilustrasi naga api di langit

Tidak akan bentrok dengan vision/lihat gambar, karena teks seperti:
- ini gambar apa
- jelaskan gambar ini
tetap diproses sebagai analisis gambar, bukan generate gambar.
