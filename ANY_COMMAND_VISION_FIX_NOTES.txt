AI LUKY Any Command Vision Fix

Masalah yang diperbaiki:
- Gambar sudah terdeteksi, tetapi caption "!ini gambar apa?" dianggap command chat biasa.
- Sekarang kalau ada gambar dan caption diawali command tidak dikenal seperti !ini, !apa, !cek, bot akan tetap menganalisis gambar.
- Command khusus tetap aman:
  !menu, !help, !remember, !clear, !search, !img, !mp3, !video, !sticker, !s, !cekmedia

Tes:
1. Kirim gambar dengan caption: !ini gambar apa?
2. Log harus masuk "Analisis gambar dengan vision model"
3. Bot harus jawab isi gambar, bukan menyuruh reverse image search.
