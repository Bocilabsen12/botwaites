AI LUKY ComfyUI Timeout Fix

Masalah:
- !animasi sudah masuk ke ComfyUI, tapi bot error:
  ComfyUI error di /history/... timeout of 30000ms exceeded
- Itu bukan selalu berarti generate gagal. Biasanya ComfyUI masih proses, tapi bot terlalu cepat timeout saat cek history.

Perbaikan:
- Timeout /history dinaikkan dari 30 detik ke 180 detik.
- Total timeout generate dinaikkan ke 30 menit.
- Timeout /prompt dan /view juga dinaikkan.
- Default Comfy dibuat ringan:
  320x320, 8 frame, 8 steps, CFG 6.
- Prompt kendaraan dirapikan agar tidak memasukkan kata manusia seperti "arms swinging naturally" untuk mobil.

Setting .env:
COMFY_TIMEOUT=1800000
COMFY_HISTORY_HTTP_TIMEOUT=180000
COMFY_PROMPT_HTTP_TIMEOUT=120000
COMFY_VIEW_HTTP_TIMEOUT=180000
COMFY_WIDTH=320
COMFY_HEIGHT=320
COMFY_FRAMES=8
COMFY_FPS=8
COMFY_STEPS=8
COMFY_CFG=6

Cara pakai:
1. Jalankan ComfyUI:
   E:
   cd \ComfyUI_windows_portable
   .\run_nvidia_gpu.bat

2. Jalankan bot:
   E:
   cd \lywbu-wa-ai-bot
   npm.cmd install
   npm.cmd start

3. Tes:
   !comfycek
   !promptanimasi mobil sport merah melaju di jalan tol malam hari
   !animasi mobil sport merah melaju di jalan tol malam hari

Kalau masih timeout:
- Lihat terminal ComfyUI apakah masih proses atau ada error.
- Turunkan lagi:
  COMFY_FRAMES=6
  COMFY_STEPS=6
  COMFY_WIDTH=288
  COMFY_HEIGHT=288
