AI LUKY Animasi Smart Prompt + Internet Search

Perubahan:
- Command !animasi sekarang memakai Gemma3:4b untuk merapikan prompt agar hasil animasi lebih bagus.
- Gemma membuat prompt bahasa Inggris yang cocok untuk ComfyUI AnimateDiff.
- Bot boleh cek internet untuk prompt yang butuh referensi visual:
  tokoh, tempat nyata, kendaraan tertentu, style, karakter, produk, landmark, dll.
- Search internet hanya membantu referensi visual/deskripsi, bukan menjamin hasil 100% mirip.
- Prompt dibuat fokus ke motion:
  orang berjalan = full body, visible legs, natural walking gait, arms swinging
  mobil/truk = vehicle moves forward, wheels rotating, road motion, motion blur
- Ditambah command:
  !promptanimasi <prompt>
  untuk melihat prompt final dari Gemma tanpa generate video.

Contoh:
!promptanimasi orang berjalan di jalan kota
!animasi orang berjalan di trotoar malam hari
!animasi mobil besar melaju kencang di jalan tol
!animasi Jokowi berjalan di panggung, cinematic

Setting .env:
CHAT_MODEL=gemma3:4b
SMART_VIDEO_PROMPT=true
VIDEO_PROMPT_SEARCH=true

Syarat:
- Ollama harus jalan.
- Model gemma3:4b harus ada:
  ollama pull gemma3:4b
- ComfyUI harus jalan.
- workflows/animatediff-api.json harus API format yang benar.
